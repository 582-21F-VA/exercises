# Exercise: Forms

First, create a form that allows users to input the year and title of a
movie. Both of these values are required to submit the form. Once the
form has been submitted, prevent the browser from sending the resquest.
Instead, store the movie in an array, and use JavaScript to render the
list of movies on the page.

Second, create a form that allows users to filter the list of movies
based on a search query. As users type in the search field, the list is
updated to display only movies whose title or year include the current
value of the field.
